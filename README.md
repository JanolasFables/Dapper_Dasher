# Dapper_Dasher

## Goal
- Roll the marble through the maze to the end as fast as possible
- Hop over incoming rolling objects without touching them

## Controls
- Jump : Space Bar
- Quit : Escape

## Details 
- Built with C++ using the raylib library
- Completed 05/21/22 from GaneDev.TV tutorial
